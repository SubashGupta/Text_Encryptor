Text Files such as passwords,bank account details and many more are sensitive information.
It is not suggested to store them in plain text on one's system. 
This program can be used to encrypt such kind of text files using Affine Cipher and Vignere Cipher.
The user has to give the Location of the file which has to be encrypted in the Input File Location Field. 
Similarly the Location of the file which has to be decrypted in the Output File Location Field. 
The Password is the key for the encryption and decryption which the user has to provide while encryption and decryption.
The keys required for the encryption and decryption are deived from the Password. 
The same password must be given to decrypt an encrypted file.
